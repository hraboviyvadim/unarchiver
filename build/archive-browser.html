The Archive Browser

Quickly view archive contents

The Archive Browser lets you easily peek into the archive, find the files you need, preview and extract them without unarchiving everything. Quick Look any file or folder and open them right from the archive

Download
  • Mac App Store
View info on archives and their contents
You can see when the file was last accessed and modified, size and compression rate, etc.
View archives of any format
Including Zip, RAR, 7-zip, Tar, Gzip and Bzip2, StuffIt, DiskDoubler, LZH, ISO, BIN, MDF, and even EXE.
Read encoded names in various characters
The Archive Browser will correctly display any non-Latin alphabet in the name of the archive

How to use The Archive Browser

1
Download the zip file
2
Unzip it, you can use the free Unarchiver for the job
3
Right click on any archive you need to view
4
Choose Open with -> The Archive Browser
5
Enjoy the view!

You can click any file inside the archive to get additional info and Quick Look or open it with Preview.
